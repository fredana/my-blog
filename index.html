MY MEMORABLE JOURNEY INTO IT WORLD

How Everything Started

My passion for the tech world started more than 15 years ago, around 2001. I was in 4th year of high school. Where I come from (West Africa), IT was not quite developed. So, this dream of understanding how these computers worked stayed with me until around 2007 when I started college. But unfortunately, I could not concretize it...

It was only a few years after my arrival in the USA that I was able to fully "throw my feet together" in the field of tech and begin to really learn the basic components of a web page, in particular HTML, CSS, and JAVASCRIPT ( the three main "ingredients" of a web page presentation).

In the following lines, we will see in a basic way what importance these two major "ingredients" represent and what roles they play in the creation of a web page.

Definitions

A webpage is like a human body structure:

  • The skeleton: HTML
  • Flesh/Clothing: CSS
  • The functioning of the organs: JAVASCRIPT

  1. HTML

    2. The HyperText Markup Language (HTML) is the code that is used to structure a web page and its content. It is composed of three main elements:

    • Html: contains the entire HTML document.
    • Head: contains any metadata describing the document.
    • Body: is the second child of the "html" element. it is what is displayed on a web page when opened. Everything that we want the view to see is coded into the "body" section (called opening and closing tags "<", and "/>")
    • Footer: is the section used to add a note at the bottom of a page, but it is not a "must use" element
